Cloud Resources

There are supplementary cloud resources offered by the NeCTAR Research Cloud that you may wish to use. Each of these resources has its own quota that you can apply for.
Volume storage A persistent volume looks and acts like a hard drive that can be attached to your virtual machines. The data on it will persist between runs of your virtual machine. Specify the total size in gigabytes that the project will need. More information is available on the Introduction to Cloud Storage support page.
Object storage Object Storage is a large accessible online storage location that you can reach from most machines with internet connectivity. Object Storage requires cloud native applications for access. Specify the total size in gigabytes that the project will need. More information is available on the Object Storage support page.
Database The Nectar database service provides a simple interface for provisioning and managing database engines supporting MySQL, with PostgreSQL and MongoDB coming soon. You can specify the number of database servers, the total database storage your project would require, or both. More information is available on the Database service support page.
